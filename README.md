# Sistema de Gestão para Projeto Social Esportivo

Sistema web desenvolvido para gerenciamento de projetos sociais esportivos.

## Tecnologias Utilizadas

- Python
- Flask
- SQLite
- Bootstrap
- JavaScript
- Chart.js
- Pandas
- Pytest

## Funcionalidades

- Login e logout
- Controle de permissões
- Dashboard com gráficos
- CRUD de modalidades
- CRUD de participantes
- CRUD de usuários
- Relatórios
- Exportação Excel
- Testes automatizados
